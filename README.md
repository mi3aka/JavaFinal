# 项目结构

`FrontEnd.java`swing渲染

`MysqlOperate.java`数据库操作

`NewJTable.java`将JTable的部分函数重载

`ReadFile.java`读取`user.txt`并导入数据库中

`docker-compose.yml`docker-compose编排,用于启动mysql

`init.sql`定义数据库结构,用于docker的初始化
